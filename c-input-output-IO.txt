C Output —
In C programming, printf() is one of the main output function.

Ex,

#include <stdio.h>
void main()
{
    printf("Hello world !");
}



C Input —
In C programming, scanf() is one of the commonly used function to take input from the user.

Ex,

#include <stdio.h>
int val;
void main()
{
    printf("Enter an integer: ");
    scanf("%d", &val);
    printf("You've entered = %d", val);
}
